## Component Breakdown

### StatCard Component
```
┌──────────────────┐
│   ⭐ (gradient)  │ ← Icon with gradient bg
│                  │
│   1,250          │ ← Large value
│   +15% ↗        │ ← Trend indicator
│                  │
│ Loyalty Points   │ ← Label
└──────────────────┘
     ↑ Hover: Lifts up
     ↑ Decorative gradient blob
```

### Activity Item
```
┌──────────────────────────────────┐
│ 📦  Order #123            2h ago │
│     Order placed                 │
│     $45.99                       │
└──────────────────────────────────┘
 ↑ Icon  ↑ Title  ↑ Description  ↑ Time
```

```
┌──────────────┐

│              │
│ Order Now    │ ← Label
│ Browse menu  │ ← Description
└──────────────┘
    ↑ Hover: Lifts + icon scales

```

## Color Coding

### Stat Cards
```
⭐ Points   → Pink gradient (fuchsia-500 to pink-600)
🛍️ Orders   → Blue gradient (blue-500 to blue-600)
💰 Spent    → Green gradient (emerald-500 to emerald-600)
🎁 Rewards  → Purple gradient (purple-500 to purple-600)
```

### Activity Types
```
📦 Order Placed     → Blue
✅ Order Delivered  → Green
❌ Order Cancelled  → Red
⭐ Points Added     → Amber
🎁 Reward Earned    → Purple
```

### Quick Actions
```
🍽️ Order    → Pink
📦 Track    → Blue
❤️ Favorites → Red
🎁 Rewards  → Purple
```

## Interaction States

### Stat Card Hover
```
Before:              After Hover:
┌──────────┐        ┌──────────┐
│   ⭐      │        │   ⭐      │ ← Lifts -4px
│   1,250  │   →    │   1,250  │ ← Border glow
│ Loyalty  │        │ Loyalty  │ ← Cursor pointer
└──────────┘        └──────────┘
```

### Quick Action Hover
```
Before:              After Hover:
┌──────────┐        ┌──────────┐
│ 🍽️        │        │ 🍽️ (1.1x) │ ← Icon scales
│ Order    │   →    │ Order → │ ← Arrow appears
└──────────┘        └──────────┘ ← Lifts -4px
```

### Button Tap
```
Scale: 1.0 → 0.98 → 1.0
Duration: 150ms
```

## Animations Timeline

```
Page Load:
  0ms    │ Container starts (opacity 0)
  100ms  │ Hero fades in ↑
  200ms  │ First stat card ↑
  300ms  │ Second stat card ↑
  400ms  │ Third stat card ↑
  500ms  │ Fourth stat card ↑
  600ms  │ Quick actions ↑
  700ms  │ Activity feed ↑
         │ Sidebar ↑
  1200ms │ Progress bar fills →
```

```
Ongoing:
  Every 3s │ Food emoji floats ↑↓
  On hover │ Cards lift up
  On tap   │ Cards scale down
```
